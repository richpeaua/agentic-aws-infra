# task-queue

An SQS work queue with a companion dead-letter queue.
Messages that fail processing more than five times (`maxReceiveCount = 5`) are redriven from the main queue to the DLQ, so poison messages are isolated for inspection instead of blocking the queue.
Both queue URLs and ARNs are exported (the DLQ ARN so alarms can watch DLQ depth).
